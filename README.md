# Project 
This was motivated by a news article where dogs were kidnapped and had to figure out a way back home. 
Maybe this could help owners adjust to the growing advantages of AI and how they can use it to  train their pets.

# RoboNav
An RL-based autonomous navigation system where an agent learns 
to navigate unknown environments through trial and error,
trained entirely in simulation using ROS2, Gazebo, and PPO.

## The Story
Rex the dog wakes up lost in an unfamiliar neighborhood. 
He has no map. Only his senses. Through thousands of attempts, 
he learns to find his way home.

## Tech Stack
- ROS2 Humble
- Gazebo Fortress
- Stable Baselines3 (PPO, SAC, TD3)
- Custom Gymnasium Environment
- FastAPI + WebSockets (live dashboard)
- Docker
- AWS
